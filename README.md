Automation Testing Project
Project Overview:
This project automates test cases for the website https://practice.automationtesting.in/ to validate the following scenarios:

Asserting the existence of the "Thinking in HTML" book with its price
Adding the book to the cart
Navigating to the shopping cart
Verifying the added item with its details
Proceeding to checkout
Asserting the display of the Billing Details form
Technology Stack:

Java
Selenium WebDriver
TestNG
Page Object Model
Maven

Prerequisites:

Java Development Kit (JDK)
Apache Maven
A preferred IDE IntelliJ IDEA
WebDriver for your desired browser  ChromeDriver
